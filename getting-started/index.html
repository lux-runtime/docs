    Getting Started¶

    Requirements¶

    • Rust 1.70 or later
    • Git

    Build from Source¶

    git clone https://github.com/lux-runtime/lux
cd lux/lux-runtime
cargo build --release

    The binaries will be at:

    Component Windows Linux/macOS
    Lux Runtime target\release\lux.exe target/release/lux

    Build LPM (Package Manager)¶

    cd lux/lpm
cargo build --release

    Adding to PATH¶

    # Create installation directory
New-Item -ItemType Directory -Path "$env:USERPROFILE\.lux\bin" -Force

# Copy binaries
Copy-Item "lux-runtime\target\release\lux.exe" -Destination "$env:USERPROFILE\.lux\bin\"
Copy-Item "lpm\target\release\lpm.exe" -Destination "$env:USERPROFILE\.lux\bin\"

# Add to PATH (permanent - requires restart of terminal)
$currentPath = [Environment]::GetEnvironmentVariable("Path", "User")
[Environment]::SetEnvironmentVariable("Path", "$currentPath;$env:USERPROFILE\.lux\bin", "User")

    # Copy to local bin
mkdir -p ~/.local/bin
cp lux-runtime/target/release/lux ~/.local/bin/
cp lpm/target/release/lpm ~/.local/bin/

# Add to PATH (add to ~/.bashrc or ~/.zshrc)
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

    Verify Installation¶

    lux --version
lpm --version

    Create Your First Project¶

    # Initialize a new project
lpm init

# Follow the prompts:
# - Package name
# - Version
# - Description
# - Author
# - License
# - Entry point

# Run your project
lpm run dev

    Project Structure¶

    After lpm init, your project looks like:

    my-project/
├── .lux/
│   └── types/          # Type definitions for IDE
│       ├── fs.d.luau
│       ├── process.d.luau
│       ├── task.d.luau
│       ├── serde.d.luau
│       ├── regex.d.luau
│       ├── datetime.d.luau
│       ├── stdio.d.luau
│       ├── luau.d.luau
│       └── ffi.d.luau
├── .vscode/
│   └── settings.json   # VS Code config for Luau LSP
├── src/
│   └── main.luau       # Entry point
├── .gitignore
├── .luaurc             # Luau language config
└── lux.toml            # Project manifest

    IDE Setup¶

    VS Code¶

    1. Install the Luau Language Server extension
    2. lpm init automatically configures .vscode/settings.json
    3. Autocomplete works for require("@lux/...")

    Other Editors¶

    Create .luaurc with:

    {
    "languageMode": "strict",
    "aliases": {
        "lux": ".lux/types"
    }
}
