    Globals¶

    Global variables and types available in the Lux runtime.

    Core Globals¶

    _G¶

    A table shared across the entire environment, used for global data storage.

    _G: { [any]: any }

    _VERSION¶

    The current Lux runtime version string.

    _VERSION: string

    print¶

    Prints values to stdout.

    print(...: any) -> ()

    warn¶

    Prints values to stderr (or structured warning output).

    warn(...: any) -> ()

    require¶

    Loads a module.

    require(path: string) -> any

    error¶

    Throws an error.

    error(message: any, level: number?) -> never

    pcall / ypcall¶

    Calls a function in protected mode (catches errors). ypcall is an alias for pcall in Luau.

    pcall<T..., R...>(f: (T...) -> R..., ...: T...) -> (boolean, R...)

    xpcall¶

    Calls a function with a custom error handler.

    xpcall<T..., R...>(f: (T...) -> R..., err: (string) -> any, ...: T...) -> (boolean, R...)

    type / typeof¶

    Returns the type of a value. typeof returns more granular types for Lux objects (e.g., "Vector3", "DateTime").

    type(v: any) -> string
typeof(v: any) -> string

    Data Types¶

    Lux includes data types compatible with Roblox API for easier portability.

    Color3¶

    RGB color value (0-1 range).

    type Color3 = { R: number, G: number, B: number }

    Constructors:

    Function Description
    Color3.new(r, g, b) Create from 0-1 values
    Color3.fromRGB(r, g, b) Create from 0-255 values
    Color3.fromHSV(h, s, v) Create from HSV
    Color3.fromHex(hex) Create from hex string

    Methods:

    Method Returns Description
    :Lerp(goal, alpha) Color3 Interpolate between colors
    :ToHSV() h, s, v Convert to HSV
    :ToHex() string Convert to hex ("FF0000")

    Vector2¶

    2D vector for positions and directions.

    type Vector2 = { X: number, Y: number, Magnitude: number, Unit: Vector2 }

    Constructors:

    • Vector2.new(x, y) - Create from components
    • Vector2.zero - Constant (0, 0)
    • Vector2.one - Constant (1, 1)

    Methods:

    Method Returns Description
    :Lerp(goal, alpha) Vector2 Linear interpolation
    :Dot(other) number Dot product
    :Cross(other) number 2D cross (z-component)

    Operators: +, -, *, /, - (unary), ==

    Vector3¶

    3D vector for positions and directions.

    type Vector3 = { X: number, Y: number, Z: number, Magnitude: number, Unit: Vector3 }

    Constructors:

    • Vector3.new(x, y, z) - Create from components
    • Vector3.zero - Constant (0, 0, 0)
    • Vector3.one - Constant (1, 1, 1)

    Methods:

    Method Returns Description
    :Lerp(goal, alpha) Vector3 Linear interpolation
    :Dot(other) number Dot product
    :Cross(other) Vector3 Cross product (perpendicular)

    Operators: +, -, *, /, - (unary), ==

    UDim¶

    UI Dimension with scale and offset.

    type UDim = { Scale: number, Offset: number }

    Final size = parentSize * Scale + Offset

    Constructors:

    • UDim.new(scale, offset)

    UDim2¶

    2D UI Dimension for positions and sizes.

    type UDim2 = { X: UDim, Y: UDim }

    Constructors:

    Function Description
    UDim2.new(xScale, xOffset, yScale, yOffset) Full constructor
    UDim2.fromScale(x, y) Scale only (offset = 0)
    UDim2.fromOffset(x, y) Offset only (scale = 0)

    Rect¶

    2D Rectangle.

    type Rect = { Width: number, Height: number }

    Constructors:

    • Rect.new(minX, minY, maxX, maxY)

    NumberRange¶

    A range between two numbers.

    type NumberRange = { Min: number, Max: number }

    Constructors:

    • NumberRange.new(min, max) - Coordinates auto-normalize

    Enum¶

    Enumeration types for type-safe constants.

    Enum.KeyCode¶

    Platform-specific keyboard key codes for FFI input handling.

    • Windows: VK_* codes (for GetAsyncKeyState, etc.)
    • Linux: evdev KEY_* codes
    • macOS: Carbon kVK_* codes
    -- FFI example
ffi.cdef("int16_t GetAsyncKeyState(int vKey);")
if ffi.C.GetAsyncKeyState(Enum.KeyCode.A) ~= 0 then
    print("A key is pressed!")
end

    Keys available: A-Z, Zero-Nine, F1-F12, Escape, Tab, Space, Return, Backspace, Delete, arrows (Up/Down/Left/Right), modifiers (LeftShift, RightShift, LeftControl, etc.), Numpad0-9, and more.

    Enum.MouseButton¶

    Name Value Description
    Left 0 Primary button
    Right 1 Secondary button
    Middle 2 Wheel click
    Button4 3 Extra button
    Button5 4 Extra button

    Back 9 Slight overshoot
    Bounce 10 Bouncing
