    datetime¶

    Date and time operations.

    local datetime = require("@lux/datetime")

    Constructors¶

    now¶

    Returns current date and time.

    datetime.now() -> DateTime

    fromUnixTimestamp¶

    Creates DateTime from Unix timestamp (seconds). Fractional part represents milliseconds/nanoseconds.

    datetime.fromUnixTimestamp(unixTimestamp: number) -> DateTime

    fromUniversalTime¶

    Creates DateTime from UTC time components.

    datetime.fromUniversalTime(values: DateTimeValues) -> DateTime

    fromLocalTime¶

    Creates DateTime from local time components.

    datetime.fromLocalTime(values: DateTimeValues) -> DateTime

    fromIsoDate¶

    Creates DateTime from ISO 8601 / RFC 3339 string.

    datetime.fromIsoDate(isoDate: string) -> DateTime

    DateTime Properties¶

    DateTime.unixTimestamp: number       -- Seconds since Unix epoch
DateTime.unixTimestampMillis: number -- Milliseconds since Unix epoch

    DateTime Methods¶

    toUniversalTime¶

    Extracts UTC time components.

    DateTime:toUniversalTime() -> DateTimeValues

    toLocalTime¶

    Extracts local time components.

    DateTime:toLocalTime() -> DateTimeValues

    formatUniversalTime¶

    Formats as UTC using strftime pattern.

    DateTime:formatUniversalTime(formatString: string?, locale: Locale?) -> string

    formatLocalTime¶

    Formats as local time using strftime pattern.

    DateTime:formatLocalTime(formatString: string?, locale: Locale?) -> string

    toIsoDate¶

    Returns ISO 8601 string.

    DateTime:toIsoDate() -> string

    Types¶

    DateTimeValues: 

    type DateTimeValues = {
    year: number,        -- 1400 -> 9999
    month: number,       -- 1 -> 12
    day: number,         -- 1 -> 31
    hour: number,        -- 0 -> 23
    minute: number,      -- 0 -> 59
    second: number,      -- 0 -> 60 (60 = leap second)
    millisecond: number?, -- 0 -> 999 (optional for input, always present in output)
}

    Locale: 

    type Locale = "en" | "de" | "es" | "fr" | "it" | "ja" | "pl" | "pt-br" | "pt" | "tr"

    Format Tokens¶

    Token Example Description
    %Y 2024 Year (4 digits)
    %m 12 Month (01-12)
    %d 25 Day (01-31)
    %H 15 Hour (00-23)
    %M 30 Minute (00-59)
    %S 45 Second (00-59)
    %A Friday Weekday name
    %B December Month name

    See chrono documentation for all tokens.

    Examples¶

    local datetime = require("@lux/datetime")

-- Current time
local now = datetime.now()
print(now.unixTimestamp)

-- Formatting
print(now:toRfc3339())             -- 2024-12-25T15:30:00Z
print(now:toRfc2822())             -- Wed, 25 Dec 2024 15:30:00 +0000
print(now:formatLocalTime("%Y-%m-%d %H:%M:%S"))
print(now:formatLocalTime("%A, %d %B %Y", "fr")) -- French locale

-- From timestamp
local past = datetime.fromUnixTimestamp(0)
print(past:toRfc3339()) -- 1970-01-01T00:00:00Z

-- From components
local custom = datetime.fromLocalTime({
    year = 2024, month = 12, day = 25,
    hour = 12, minute = 0, second = 0
})

-- Extract components
local parts = now:toLocalTime()
print(parts.year, parts.month, parts.day)
print(parts.hour, parts.minute, parts.second)

-- Parse RFC 3339
local parsed = datetime.fromRfc3339("2024-12-25T12:00:00Z")
